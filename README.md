# Kopet - Single Page Website

React (Vite)를 사용한 단일 페이지 정보 및 소셜 링크 웹사이트입니다.

## 주요 기능

- 헤더: 'kopet' 제목
- 이미지 슬라이드쇼: dogphoto 폴더의 사진들이 1.5초 간격으로 자동 전환
- 소개글: 영어/태국어 소개글
- 소셜 링크: Instagram, Tiktok, Line 링크

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```
